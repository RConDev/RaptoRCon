RaptoRCon

An alternative RCon Administration Tool written in C#

Download Sample Release v0.2 Download this project as a .zip file
keyboard-centric bug tracker continuous integration server

Welcome to RaptoRCon.

This project aims to be a real alternative for ProCon.

Designed for supporting multiple games, a modern web based infrastructure, but yet as powerful as known administration tools, this project is hopefully supported by many (plugin-)developers, admins, yet using and developing for ProCon.

Your support is needed.

Your support is definitly needed. Wheather you can provide experience in programming, adminstrating game servers or just a financial support is highly needed.

As you might have notice within the project's repository, the license used is MIT-License. So this tool is free of charge, and will be.

Fork the project and start coding, discuss on the issues, provide your ideas,... do as you like, and maybe you will become one of us

If you like this project and it's aims, but don't have the knowledge of programming or administration, just provide your financial support via PayPal.
